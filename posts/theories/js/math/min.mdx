### [Math.min()](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Math/min)

- Math.min() 함수는 주어진 숫자들 중 가장 작은 값을 반환해요.

```js
Math.min([value1[, value2[, ...]]])
```

```js
Math.min(2, 3, 1); // 1
```

구조 분해 할당을 사용하여 배열의 요소 중 가장 작은 값을 찾을 수 있어요.

```js
const arr = [2, 3, 1];
Math.min(...arr); // 1
```
