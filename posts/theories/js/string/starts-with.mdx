### [String.prototype.startsWith()](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/String/startsWith)

`startsWith(){:js}` 메서드는 어떤 문자열이 특정 문자로 시작하는지 확인하여 결과를 `true` 혹은 `false`로 반환해요.

```js
startsWith(searchString);
startsWith(searchString, position);
```

- `searchString` : 찾고자 하는 문자열
- `position` : `searchString`을 탐색할 위치, 기본값은 `0`

```js
const str1 = 'Saturday night plans';

console.log(str1.startsWith('Sat'));
// Expected output: true

console.log(str1.startsWith('Sat', 3));
// Expected output: false
```
