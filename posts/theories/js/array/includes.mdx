### [Array.prototype.includes()](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/includes)

`includes(){:js}` 메서드는 배열의 항목에 특정 값이 포함되어 있는지를 판단하여 적절히 `true` 또는 `false`를 반환해요.

```js
includes(searchElement[, fromIndex])
```

- `searchElement` : 배열에서 찾을 값
- `fromIndex` : 검색을 시작할 인덱스. 음수를 지정하면 배열의 끝에서부터 검색을 시작해요. 기본값은 `0`이에요.

```js
const array1 = [1, 2, 3];

console.log(array1.includes(2));
// Expected output: true

const pets = ['cat', 'dog', 'bat'];

console.log(pets.includes('cat'));
// Expected output: true

console.log(pets.includes('at'));
// Expected output: false
```
