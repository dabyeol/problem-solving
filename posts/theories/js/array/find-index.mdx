### [Array.prototype.findIndex()](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Global_Objects/Array/findIndex)

`findIndex(){:js}` 메서드는 주어진 판별 함수를 만족하는 배열의 첫 번째 요소에 대한 인덱스를 반환해요. 만족하는 요소가 없으면 -1을 반환해요.

```js
findIndex(callbackFn[, thisArg])
```

- callbackFn : 각 요소를 시험할 함수. 다음 세 가지 인수를 받아요.
  - element : 배열에서 처리중인 현재 요소
  - index : 배열에서 처리중인 현재 요소의 인덱스
  - array : `findIndex(){:js}` 함수가 호출된 배열
- thisArg : 선택 사항. callbackFn 함수 내부에서 `this`로 사용할 값

```js
const array1 = [5, 12, 8, 130, 44];

const isLargeNumber = element => element > 13;

console.log(array1.findIndex(isLargeNumber));
// Expected output: 3
```
