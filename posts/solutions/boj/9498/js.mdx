## 이론

[**A+B (1000)**](../1000/js)를 먼저 풀어보세요!

### 입력받기

```js {2}
const fs = require('fs');
const input = Number(fs.readFileSync('/dev/stdin'));
```

`Number(){:js}`를 사용하여 입력받은 문자열을 숫자로 변환해요.

## 코드

```js
const fs = require('fs');
const input = Number(fs.readFileSync('/dev/stdin'));

if (input >= 90) {
  console.log('A');
} else if (input >= 80) {
  console.log('B');
} else if (input >= 70) {
  console.log('C');
} else if (input >= 60) {
  console.log('D');
} else {
  console.log('F');
}
```
