### 입력받기

한 줄에 하나씩 입력받는 방법에 대해 알아봐요.

```js {2-3}
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().split('\n');
const [x, y] = input.map(Number);
```

`split(){:js}` 함수를 사용하여 줄바꿈 문자를 기준으로 문자열을 나누어 배열로 만들면 돼요.

## 코드

```js
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().split('\n');
const [x, y] = input.map(Number);

if (x > 0 && y > 0) {
  console.log(1);
} else if (x < 0 && y > 0) {
  console.log(2);
} else if (x < 0 && y < 0) {
  console.log(3);
} else if (x > 0 && y < 0) {
  console.log(4);
}
```
