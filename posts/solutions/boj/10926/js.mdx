### 입력받기

```js {2}
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().trim();
```

`trim(){:js}` 함수는 문자열의 앞뒤 공백(띄어쓰기, 탭, 줄바꿈 등)을 제거해요.
사용하지 않으면 입력받은 문자열의 앞뒤에 공백이 있을 수 있어 틀렸다고 나올 수 있어요.

## 코드

```js
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().trim();

console.log(input + '??!');
```
