### 특수문자 출력

특수문자를 출력하기 위해서는 `\`를 앞에 붙여줘야 해요.

```js
console.log('\\'); // \
```

`'`와 `"`도 특수문자로 취급되는데, 이 두 문자를 출력하기 위해서는 위의 방법을 사용하거나, 아래와 같이 작은 따옴표와 큰 따옴표를 번갈아 사용해야 해요.

```js
console.log("'"); // '
console.log('"'); // "
```

## 코드

```js
console.log('|\\_/|');
console.log('|q p|   /}');
console.log('( 0 )"""\\');
console.log('|"^"`    |');
console.log('||_/=\\\\__|');
```
