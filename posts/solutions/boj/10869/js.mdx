## 이론

[**A+B (1000)**](../1000/js)를 먼저 풀어보세요!

### 몫과 나머지

- 몫: `Math.floor(a / b){:js}`
- 나머지: `a % b{:js}`

`Math.floor(){:js}`는 소수점 이하를 버리는 함수예요.

## 코드

```js
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().split(' ');
const [a, b] = input.map(Number);

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(Math.floor(a / b));
console.log(a % b);
```
