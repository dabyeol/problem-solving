## 이론

### 입력받기

```js {2}
const fs = require('fs');
const input = Number(fs.readFileSync(0));
```

`Number(){:js}`를 사용하여 입력받은 문자열을 숫자로 변환해요.

### 출력하기

```js
console.log(
  Number(input % 4 === 0 && (input % 100 !== 0 || input % 400 === 0))
);
```

boolean 값을 숫자로 변환하여 출력해요.
자바스크립트에서는 `==={:js}` 연산자를 사용하여 값과 타입을 비교해요.
`=={:js}` 연산자는 값만 비교하고 타입은 비교하지 않아요.
