## 이론

`map(){:js}` 함수를 이용해서 먼저 number를 숫자 배열로 바꿔요. 그리고 `reduce(){:js}` 함수를 이용해서 배열의 합을 구하고 9로 나눈 나머지를 구해요.

`reduce(){:js}` 함수 안에서 `splice(){:js}` 함수를 이용해서 배열 내용을 변경해요. `reverse(){:js}` 함수를 이용해서 배열을 뒤집어요.

## 코드

```js
function solution(my_string, queries) {
  return queries
    .reduce(
      (acc, [s, e]) => {
        acc.splice(s, e - s + 1, ...acc.slice(s, e + 1).reverse());

        return acc;
      },
      [...my_string]
    )
    .join('');
}
```
