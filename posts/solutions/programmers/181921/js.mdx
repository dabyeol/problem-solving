## 이론

```js {4-6}
function solution(l, r) {
  var answer = [];
  for (let i = l; i <= r; i++) {
    if ([...String(i)].every(c => c === '0' || c === '5')) {
      answer.push(i);
    }
  }

  return answer.length ? answer : [-1];
}
```

`every(){:js}` 메서드를 사용하면 주어진 조건을 만족하는지 확인할 수 있어요.

이 코드를 정규표현식으로 바꾸면 아래와 같아요.

```js {4-6}
function solution(l, r) {
  var answer = [];
  for (let i = l; i <= r; i++) {
    if (String(i).match(/^[05]+$/)) {
      answer.push(i);
    }
  }

  return answer.length ? answer : [-1];
}
```
