먼저 각각의 숫자가 담긴 배열을 만들어요.

```js
[...age.toString()];
```

그리고 이 배열을 순회하면서 알파벳으로 변환해요.

```js
[...age.toString()].map(n => 'abcdefghij'[n]);
```

문자열도 배열처럼 인덱스로 접근할 수 있어요. 그래서 각각의 숫자를 알파벳으로 변환할 수 있어요.

> `n`이 문자열이지만 문자열을 인덱스로 접근하면 자동으로 숫자로 변환해요.
>
> ```js
> 'abcdefghij'[0]; // 'a'
> 'abcdefghij'['0']; // 'a'
> ```

마지막으로 이 배열을 문자열로 만들어요.

```js
[...age.toString()].map(n => 'abcdefghij'[n]).join('');
```
