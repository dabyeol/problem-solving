`sides`의 원소 중 작은 수를 `a`, 큰 수를 `b`, 나머지를 `c`라고 해볼게요.

- 가장 긴 변의 길이가 `b`인 경우
  `c <= b < a + c{:js}`를 만족하면 삼각형이 성립해요.
  따라서 `0 <= b - c < a{:js}`, `-b <= -c < a - b{:js}`, `b - a < c <= b{:js}`를 만족하면 삼각형이 성립해요.
  이를 만족하는 `c`의 개수는 `a`개예요.

- 가장 긴 변이 `c`인 경우
  `b < c < a + b{:js}`를 만족하면 삼각형이 성립해요.
  이를 만족하는 `c`의 개수는 `a - 1{:js}`개예요.

따라서 `2 * a - 1{:js}`이 답이에요.
