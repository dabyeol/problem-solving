## 이론

### 문제 분석

세 숫자가 모두 다르거나(첫 번째 규칙) 세 숫자가 모두 같은 경우(세 번째 규칙)가 아니라면 두 번째 규칙을 만족해요.
복잡한 조건문을 사용하지 않고 풀 수 있어요.

## 코드

```js
function solution(a, b, c) {
  if (a !== b && a !== c && b !== c) {
    return a + b + c;
  }

  if (a === b && a === c) {
    return (a + b + c) * (a * a + b * b + c * c) * (a ** 3 + b ** 3 + c ** 3);
  }

  return (a + b + c) * (a ** 2 + b ** 2 + c ** 2);
}
```
