## 이론

원소를 1로 만들기 위해서 필요한 나누기 연산의 횟수는 원소를 2진수로 변환한 수의 자릿수 - 1과 같아요.
`toString(2){:js}`를 통해 2진수로 변환할 수 있어요.

<Reduce />

## 코드

```js
function solution(num_list) {
  return num_list.reduce((acc, cur) => acc + cur.toString(2).length - 1, 0);
}
```
