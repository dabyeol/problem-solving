## 이론

### 반복문(for)

for문은 반복문의 가장 기본이 되는 문법이예요.
for문은 3가지의 인자를 받아요.

- 첫번째 인자는 반복문이 시작할 때 실행되는 부분이에요.
- 두번째 인자는 반복문이 실행될 조건을 작성하는 부분이에요.
- 세번째 인자는 반복문이 한번 실행된 후 실행되는 부분이에요.

```js
for (let i = 0; i < 10; i++) {
  console.log(i);
}
```

위의 코드는 0부터 9까지의 숫자를 출력하는 코드예요.

홀수만 출력하고 싶다면 아래와 같이 작성하면 되요.

```js
for (let i = 1; i < 10; i += 2) {
  console.log(i);
}
```

### 제곱

제곱은 `**{:js}` 연산자를 사용하면 되요.

```js
console.log(2 ** 2); // 4
console.log(2 ** 3); // 8
console.log(2 ** 4); // 16
```

## 코드

```js
function solution(n) {
  var answer = 0;
  if (n % 2) {
    for (let i = 1; i <= n; i += 2) {
      answer += i;
    }
  } else {
    for (let i = 2; i <= n; i += 2) {
      answer += i ** 2;
    }
  }

  return answer;
}
```
