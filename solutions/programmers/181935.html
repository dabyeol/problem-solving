코딩 기초 트레이닝

홀짝에 따라 다른 값 반환하기

문제 설명

양의 정수 nn이 매개변수로 주어질 때, nn이 홀수라면 nn 이하의 홀수인 모든 양의 정수의 합을 return 하고 nn이 짝수라면 nn 이하의 짝수인 모든 양의 정수의 제곱의 합을 return 하는 solution 함수를 작성해 주세요.

제한사항

  • 1 ≤ nn ≤ 100

입출력 예

nresult
716
10220

입출력 예 설명

입출력 예 #1

  • 예제 1번의 nn은 7로 홀수입니다. 7 이하의 모든 양의 홀수는 1, 3, 5, 7이고 이들의 합인 1 + 3 + 5 + 7 = 16을 return 합니다.

입출력 예 #2

  • 예제 2번의 nn은 10으로 짝수입니다. 10 이하의 모든 양의 짝수는 2, 4, 6, 8, 10이고 이들의 제곱의 합인 22 + 42 + 62 + 82 + 102 = 4 + 16 + 36 + 64 + 100 = 220을 return 합니다.
