문자열 겹쳐쓰기

문제 설명

문자열 my_stringmy_string, overwrite_stringoverwrite_string과 정수 ss가 주어집니다. 문자열 my_stringmy_string의 인덱스 ss부터 overwrite_stringoverwrite_string의 길이만큼을 문자열 overwrite_stringoverwrite_string으로 바꾼 문자열을 return 하는 solution 함수를 작성해 주세요.

제한사항

  • my_stringmy_string와 overwrite_stringoverwrite_string은 숫자와 알파벳으로 이루어져 있습니다.
  • 1 ≤ overwrite_stringoverwrite_string의 길이 ≤ my_stringmy_string의 길이 ≤ 1,000
  • 0 ≤ ss ≤ my_stringmy_string의 길이 - overwrite_stringoverwrite_string의 길이

입출력 예

my_stringoverwrite_stringsresult
"He11oWor1d""lloWorl"2"HelloWorld"
"Program29b8UYP""merS123"7"ProgrammerS123"

입출력 예 설명

입출력 예 #1

  • 예제 1번의 my_stringmy_string에서 인덱스 2부터 overwrite_stringoverwrite_string의 길이만큼에 해당하는 부분은 "11oWor1"이고 이를 "lloWorl"로 바꾼 "HelloWorld"를 return 합니다.

입출력 예 #2

  • 예제 2번의 my_stringmy_string에서 인덱스 7부터 overwrite_stringoverwrite_string의 길이만큼에 해당하는 부분은 "29b8UYP"이고 이를 "merS123"로 바꾼 "ProgrammerS123"를 return 합니다.
