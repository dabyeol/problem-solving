코딩 기초 트레이닝

두 수의 연산값 비교하기

문제 설명

연산 ⊕는 두 정수에 대한 연산으로 두 정수를 붙여서 쓴 값을 반환합니다. 예를 들면 다음과 같습니다.

  • 12 ⊕ 3 = 123
  • 3 ⊕ 12 = 312

양의 정수 aa와 bb가 주어졌을 때, aa ⊕ bb와 2 * a * b2 * a * b 중 더 큰 값을 return하는 solution 함수를 완성해 주세요.

단, aa ⊕ bb와 2 * a * b2 * a * b가 같으면 aa ⊕ bb를 return 합니다.

제한사항

  • 1 ≤ aa, bb < 10,000

입출력 예

abresult
291364
912912

입출력 예 설명

입출력 예 #1

  • aa ⊕ bb = 291 이고, 2 * a * b2 * a * b = 364 입니다. 둘 중 더 큰 값은 364 이므로 364를 return 합니다.

입출력 예 #2

  • aa ⊕ bb = 912 이고, 2 * a * b2 * a * b = 364 입니다. 둘 중 더 큰 값은 912 이므로 912를 return 합니다.
